## **LATIHAN MENULIS BAHASA ARAB (MAHARAH AL‑KITABAH)**

**Petunjuk Umum:**

1. Bacalah setiap latihan dengan teliti.
2. Salin kata atau kalimat di kotak yang tersedia.
3. Gunakan pena yang jelas dan rapi.
4. Latihan ini bertujuan meningkatkan kemampuan menulis bahasa Arab.

---

Salin huruf hijaiyah berikut di kolom kanan!

### **Latihan 2 – Menulis Kata Sederhana**

Tuliskan kata Arab berikut dengan benar!

```
1. مدرسة   (madrasah = sekolah)  
2. كتاب    (kitab = buku)  
3. طالب    (thalib = siswa)  
4. معلم    (muʿallim = guru)  
5. مسجد    (masjid = masjid)
```
